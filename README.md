# Next.js App

This project now runs on the Next.js App Router.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the development server:
   `npm run dev`
3. Build for production:
   `npm run build`
4. Start the production server:
   `npm run start`

If you use the Gemini-backed features in the app, set `GEMINI_API_KEY` in [.env.local](.env.local).
